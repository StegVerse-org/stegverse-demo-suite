# StegVerse Runtime Commands

## Quick Start

### Unix-like shells
```bash
chmod +x stegverse
./stegverse reset
./stegverse explain
./stegverse demo
./stegverse action deploy_change
./stegverse action-receipts
```

### Windows
```powershell
stegverse.cmd reset
stegverse.cmd explain
stegverse.cmd demo
stegverse.cmd action deploy_change
stegverse.cmd action-receipts
```
